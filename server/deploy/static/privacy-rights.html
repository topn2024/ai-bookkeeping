白记用户隐私权利

更新日期：2026年1月1日  |  生效日期：2026年1月1日

根据《中华人民共和国个人信息保护法》及相关法律法规，您对自己的个人信息享有以下权利。我们高度重视您的隐私权利，并提供便捷的操作路径帮助您行使这些权利。

一、查阅权

您有权查看我们收集和存储的您的个人信息，包括账户信息、交易记录、设备信息等。

操作路径：设置 → 隐私 → 个人信息

二、更正权

当您发现我们处理的个人信息有错误或不完整时，您有权要求我们更正或补充。

操作路径：设置 → 账户 → 编辑资料

三、删除权

在以下情形下，您可以要求我们删除您的个人信息：

  • 处理目的已实现、无法实现或不再必要
  • 我们停止提供服务或服务期限届满
  • 您撤回同意
  • 我们违反法律法规或协议处理个人信息

操作路径：设置 → 账户 → 注销账户

注销账户后，我们将在30天内删除您的个人信息，法律法规另有规定的除外。

四、导出权（可携带权）

您有权获取您的个人信息副本。我们支持将您的交易记录、账户数据导出为通用格式文件。

操作路径：设置 → 数据 → 导出我的数据

五、撤回同意权

对于基于您同意而进行的个人信息处理活动，您有权随时撤回同意。撤回同意不影响撤回前基于同意已进行的个人信息处理活动的效力。

操作路径：设置 → 隐私 → 权限管理

您可以单独关闭以下权限：

  • 相机权限（拍照记账）
  • 麦克风权限（语音记账）
  • 相册权限（图片识别）
  • 应用使用数据收集

六、账户注销权

您有权注销您的白记账户。注销后：

  • 您的账户将无法登录和使用
  • 账户信息将在30天内删除
  • 交易记录将按照财务凭证保管要求处理
  • 已购买的会员服务将无法恢复

操作路径：设置 → 账户 → 注销账户

响应时限

我们将在收到您的权利请求后15个工作日内予以回复。如因特殊原因无法在规定时限内回复，我们会提前告知您延期原因及预计回复时间。

联系我们

如您需要行使上述权利或有任何隐私相关问题，请通过以下方式联系我们：

  • 隐私专员邮箱：privacy@aibook.app
  • 应用内反馈：设置 → 帮助与反馈

完整隐私政策请查阅：《白记隐私政策》
