## 八、非功能需求

### 8.1 性能要求

| 指标 | 要求 |
|------|------|
| 页面加载时间 | < 2秒 |
| API响应时间 | < 500ms (95th) |
| 并发用户数 | 支持100并发 |
| 数据导出 | 10万条数据 < 30秒 |

### 8.2 安全要求

| 项目 | 要求 |
|------|------|
| 传输加密 | 全站HTTPS |
| 密码存储 | bcrypt加密 |
| 会话管理 | JWT + Redis |
| 登录保护 | 5次失败锁定15分钟 |
| 敏感操作 | 二次确认 + 审计日志 |

### 8.3 兼容性要求

| 项目 | 要求 |
|------|------|
| 浏览器 | Chrome/Firefox/Safari/Edge最新2个版本 |
| 分辨率 | 最低1280x720 |
| 响应式 | 支持平板设备 |
