# 权利要求书

## 基于延迟满足机制的冲动消费控制方法及系统

### 独立权利要求

**权利要求1**：一种基于延迟满足机制的冲动消费控制方法，其特征在于，包括以下步骤：

S1. 冷静期触发判定步骤：
- 基于金额分级策略判定冷静期时长：100-500元触发24小时，500-1000元触发48小时，1000元以上触发72小时；
- 分类判定：必需品（餐饮、交通、医疗）不触发，可选品（服装、娱乐、电子产品）触发；
- 历史行为判定：该商户或分类近30天取消率>50%时延长50%冷静期；

S2. 冷静期记录创建步骤：
- 记录商品信息：名称、金额、分类、购买理由；
- 计算到期时间 = 当前时间 + 冷静期时长；
- 设置状态为"等待中"；

S3. 智能提醒步骤：
- 在到期前1小时发送提醒通知；
- 提供决策辅助信息：购买理由、预算剩余、历史取消率；
- 提供操作选项：立即购买、延长冷静期、取消购买；

S4. 效果量化统计步骤：
- 计算取消率 = 已取消数量 / 总数量；
- 计算节省金额 = Σ(已取消商品金额)；
- 生成分类分析：各分类的取消率。

**权利要求2**：一种基于延迟满足机制的冲动消费控制系统，其特征在于，包括：

冷静期触发模块：用于基于金额和分类判定是否触发冷静期；

冷静期管理模块：用于创建和维护冷静期记录；

智能提醒模块：用于在到期前发送提醒并提供决策辅助；

效果量化模块：用于统计取消率、节省金额等指标。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，还包括延长冷静期步骤：

- 响应于用户选择延长冷静期；
- 延长预设时长，默认24小时；
- 记录延长次数；
- 更新到期时间。

**权利要求4**：根据权利要求1所述的方法，其特征在于，还包括个性化冷静期调整：

- 计算用户自控力评分 = 历史取消率 × 100；
- 高自控力（>60分）缩短20%冷静期；
- 低自控力（<40分）延长20%冷静期。

**权利要求5**：根据权利要求4所述的方法，其特征在于，所述个性化调整还包括：

- 收入稳定性因子：收入波动大延长冷静期；
- 预算执行因子：预算超支延长冷静期；
- 时间因子：月末（21-31号）延长20%冷静期。

**权利要求6**：根据权利要求1所述的方法，其特征在于，还包括紧急购买处理：

- 响应于用户标记为"紧急"；
- 缩短50%冷静期；
- 记录紧急标记。

**权利要求7**：根据权利要求1所述的方法，其特征在于，所述效果量化还包括效果评级：

- 优秀：取消率>60%且节省金额>月收入10%；
- 良好：取消率>50%且节省金额>月收入5%；
- 一般：取消率>40%且节省金额>月收入2%；
- 较差：取消率<40%。

**权利要求8**：根据权利要求1所述的方法，其特征在于，所述智能提醒还包括：

- 提醒频率控制：每个商品仅提醒1次，避免过度打扰；
- 决策建议：基于使用状态提供继续、延长或取消建议。

**权利要求9**：根据权利要求2所述的系统，其特征在于，还包括：

- 个性化调整模块：基于用户特征自适应调整冷静期；
- 延长管理模块：处理用户延长冷静期请求；
- 可视化模块：生成效果报告和统计图表。
