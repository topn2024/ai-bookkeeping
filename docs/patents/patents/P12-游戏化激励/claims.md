# 权利要求书

## 基于行为分析的游戏化激励方法及系统

### 独立权利要求

**权利要求1**：一种基于行为分析的游戏化激励方法，其特征在于，包括以下步骤：

S1. 多维度行为分析步骤：
- 收集用户行为数据，包括记账频率、金额准确度、分类完整度等8个维度；
- 计算行为质量评分quality_score = Σ(dimension_score) / 100；
- 计算贡献值contribution = quality_score × difficulty_factor × time_decay；

S2. 自适应奖励计算步骤：
- 根据用户等级确定奖励系数：新手用户（0-30天）×1.5，进阶用户（31-90天）×1.0，专家用户（>90天）×0.8；
- 计算连续奖励加成streak_bonus = min(连续天数×0.1, 2.0)；
- 应用防疲劳因子：总贡献>10000降低10%，>50000降低20%；

S3. 成就系统追踪步骤：
- 定义四级成就：青铜（基础）、白银（进阶）、黄金（高级）、铂金（大师）；
- 追踪成就进度，解锁时发放奖励；
- 支持隐藏成就和限时成就；

S4. 异常行为检测步骤：
- 检测频率异常：1分钟内记账>5笔为可疑，1天内>100笔为作弊；
- 检测模式异常：金额完全相同（连续10笔）、时间间隔完全相同；
- 异常评分≥60时判定为作弊，扣除50%积分并暂停奖励7天。

**权利要求2**：一种基于行为分析的游戏化激励系统，其特征在于，包括：

行为分析模块：用于收集和分析8维度用户行为；

自适应奖励模块：用于根据用户等级、连续天数、防疲劳因子计算奖励；

成就系统模块：用于追踪成就进度并解锁奖励；

防刷检测模块：用于检测异常行为并实施惩罚。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，所述8维度评估包括：

- 记账频率（15分）：每日记账15分，每周3-6次10分；
- 金额准确度（15分）：精确到分15分，精确到角10分；
- 分类完整度（15分）：完整分类15分，仅一级分类10分；
- 备注详细度（10分）：详细备注10分，简单备注5分；
- 照片凭证（10分）：有照片10分；
- 标签使用（10分）：使用标签10分；
- 预算遵守（15分）：预算内15分，超支<10%为10分；
- 数据修正（10分）：主动修正10分。

**权利要求4**：根据权利要求1所述的方法，其特征在于，所述成就系统包括：

- 基础成就（青铜）：首次记账、连续3天、记账10笔；
- 进阶成就（白银）：连续7天、记账50笔、预算达成率>90%；
- 高级成就（黄金）：连续30天、记账200笔、连续3个月预算达成；
- 大师成就（铂金）：连续365天、记账1000笔、连续12个月预算达成。

**权利要求5**：根据权利要求1所述的方法，其特征在于，还包括社交激励机制：

- 分层排行榜：新手榜（0-30天）、进阶榜（31-90天）、专家榜（>90天）、好友榜；
- 排名算法：ranking_score = 0.4×总贡献 + 0.3×质量均值 + 0.2×连续天数 + 0.1×成就数量；
- 社交奖励：超越好友50分，进入前10为100分。

**权利要求6**：根据权利要求1所述的方法，其特征在于，还包括隐藏成就：

- 午夜记账者：凌晨0-6点记账；
- 旅行记账家：在5个城市记账；
- 完美主义者：连续30天无错误。

**权利要求7**：根据权利要求1所述的方法，其特征在于，还包括限时成就：

- 春节挑战：春节期间每日记账；
- 双11挑战：双11当天记账>10笔；
- 年终总结：12月完成年度回顾。

**权利要求8**：根据权利要求2所述的系统，其特征在于，还包括：

- 数据采集模块：收集用户行为数据；
- 可视化模块：展示成就、排行榜、奖励历史；
- 通知模块：发送成就解锁、排名变化、异常警告通知。
