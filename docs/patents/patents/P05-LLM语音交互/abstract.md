# 摘要

## LLM增强的四维语音交互方法及系统

本发明公开了一种LLM增强的四维语音交互方法及系统，属于人工智能和语音交互技术领域。该方法包括：四维意图识别，定义记账、查询、配置、导航四个交互维度，通过关键词检测优先识别，无法匹配时调用LLM；LLM优先混合识别，采用四层架构（LLM优先→规则兜底→模糊匹配→用户确认），平衡准确率和响应速度；多操作识别，识别"打车35，吃饭50"等包含多个操作的语句，拆分并批量执行；上下文管理，维护维度、对话和实体三类上下文，支持多轮对话。本发明解决了现有语音记账系统意图识别准确率低、响应速度慢、功能单一、无法识别多操作指令的问题，覆盖记账应用的所有交互场景，支持复杂语句的准确识别。

**关键词**：LLM；语音交互；意图识别；四维交互；多操作识别；上下文管理
