# 摘要

## 基于地理围栏的个人预算预警方法及系统

本发明公开了一种基于地理围栏的个人预算预警方法及系统，属于个人财务管理技术领域。该方法包括：本地化位置处理，所有位置数据在用户设备本地处理，不上传服务器，保护用户隐私；围栏-预算关联，将地理围栏与个人预算分类（购物、餐饮、娱乐等）进行关联；预算预警，当用户进入围栏时查询关联预算的使用情况，根据使用率生成预警消息并通过本地通知发送。本发明还包括消费场景识别、停留时间分析、异常消费检测等功能。本发明聚焦于个人日常消费的事前预警，帮助用户在进入消费区域时提前了解预算状态，做出理性消费决策，与企业费用报销、商务里程追踪等场景完全不同。

**关键词**：地理围栏；预算预警；位置隐私；消费场景；个人财务
