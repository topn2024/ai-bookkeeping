# 权利要求书

## 自适应零基预算智能管理方法及系统

### 独立权利要求

**权利要求1**：一种自适应零基预算智能管理方法，其特征在于，包括以下步骤：

S1. 零基预算分配步骤：
- 将本月收入全额分配到各个小金库，满足约束条件 Σai = I；
- 按优先级排序小金库：必需类（1.0）、重要类（0.8）、可选类（0.5）、储蓄类（0.3）；
- 优先满足高优先级小金库的最低需求，剩余金额按优先级比例分配；

S2. 季节性识别步骤：
- 分析12个月历史消费数据，计算季节性因子 si = hi / avg；
- 应用中国市场特殊因子：春节（1月）1.3、618（6月）1.2、双11（11月）1.4、双12（12月）1.2；
- 平滑处理：si' = 0.7 × si + 0.3 × si-1；

S3. 预算季节调整步骤：
- 根据季节性因子调整预算：季节预算 = 基础预算 × 季节因子；
- 提前1个月预警高峰月份；

S4. 弹性预算设置步骤：
- 提供四级弹性：严格（±5%）、适中（±15%）、宽松（±30%）、自适应；
- 自适应弹性范围计算：[B - 2σ, B + 2σ]，σ为历史标准差；

S5. 智能预算调整步骤：
- 检测持续超支：连续3个月超支>10%时建议增加预算；
- 检测持续结余：连续3个月结余>30%时建议减少预算；
- 支持小金库间预算调拨。

**权利要求2**：一种自适应零基预算智能管理系统，其特征在于，包括：

零基预算分配模块：用于将收入全额分配到各小金库，按优先级满足需求；

季节性识别模块：用于分析历史数据计算季节性因子，内置中国市场特殊因子；

弹性预算模块：用于设置四级弹性范围，支持自适应计算；

智能调整模块：用于检测持续超支或结余并生成调整建议。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，所述优先级管理包括：

- 必需类（优先级1.0）：房租、水电、交通等刚需支出；
- 重要类（优先级0.8）：餐饮、日用品等日常支出；
- 可选类（优先级0.5）：娱乐、购物等弹性支出；
- 储蓄类（优先级0.3）：应急基金、投资等储蓄支出。

**权利要求4**：根据权利要求1所述的方法，其特征在于，所述动态调整机制包括：

- 收入增加时：优先增加储蓄类预算；
- 收入减少时：优先削减可选类预算；
- 支出超标时：从其他小金库调拨资金。

**权利要求5**：根据权利要求1所述的方法，其特征在于，还包括预算健康检查步骤：

- 实时监控：消费进度 vs 时间进度；
- 提前预警：预计月末超支提前7天预警；
- 智能建议：自动生成预算调整建议。

**权利要求6**：根据权利要求1所述的方法，其特征在于，所述自适应弹性计算包括：

- 统计历史支出数据，计算标准差σ；
- 设置弹性上界：B + 2σ；
- 设置弹性下界：B - 2σ；
- 每月根据实际数据更新σ值。

**权利要求7**：根据权利要求1所述的方法，其特征在于，所述预算调拨包括：

- 检测结余小金库和超支小金库；
- 计算可调拨金额：min(结余金额, 超支金额)；
- 生成调拨建议供用户确认；
- 执行调拨并更新预算记录。

**权利要求8**：根据权利要求2所述的系统，其特征在于，还包括：

- 收入识别模块：自动识别工资、奖金等收入并触发预算分配；
- 提前预警模块：在高峰月份前1个月提醒用户准备资金；
- 历史分析模块：展示预算执行历史和季节性消费规律。
