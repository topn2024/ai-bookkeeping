# 权利要求书

## 智能自适应财务数据可视化方法及系统

### 独立权利要求

**权利要求1**：一种智能自适应财务数据可视化方法，其特征在于，包括以下步骤：

S1. 渐进式信息披露步骤：
- 将财务信息分为核心层（3-5个指标）、详细层（10-15个指标）、深度层（20+个指标）三层架构；
- 根据用户点击、停留时间（>5秒）、滚动行为决定是否展开下一层；
- 采用300ms缓动动画实现渐进式展开；

S2. 用户能力评估步骤：
- 采集使用时长、功能使用广度、高级功能使用频率、错误率四个维度数据；
- 计算用户能力评分S∈[0,100]，其中使用时长占权重30分、功能广度占30分、高级功能频率占20分、错误率占20分；

S3. 自适应复杂度调整步骤：
- 简化模式（S<30）：仅显示核心功能，隐藏高级选项，提供引导提示；
- 标准模式（30≤S<60）：显示常用功能，高级功能折叠；
- 专业模式（60≤S<80）：显示所有功能，高级功能可见；
- 专家模式（S≥80）：显示所有功能，支持快捷操作和自定义图表；

S4. 交互式数据钻取步骤：
- 支持时间维度钻取：年→月→周→日；
- 支持分类维度钻取：一级分类→二级分类→商户；
- 支持金额维度钻取：总额→分项→明细；

S5. 智能图表推荐步骤：
- 根据数据特征和用户任务自动推荐图表类型；
- 占比分析推荐饼图/环形图，趋势分析推荐折线图/面积图，对比分析推荐柱状图。

**权利要求2**：一种智能自适应财务数据可视化系统，其特征在于，包括：

渐进式披露模块：用于管理三层信息架构，控制信息展开；

能力评估模块：用于评估用户能力，生成能力评分；

复杂度调整模块：用于根据能力评分动态调整界面复杂度；

交互钻取模块：用于支持多维度数据钻取；

图表推荐模块：用于智能推荐最佳图表类型。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，所述渐进式信息披露还包括上下文感知披露：

- 根据用户当前任务调整披露内容；
- 查看预算时优先展示预算相关信息；
- 记账后优先展示新增交易的影响。

**权利要求4**：根据权利要求1所述的方法，其特征在于，所述用户能力评估的计算方式为：

- 使用时长评分：新手（<7天）0-30分、熟练（7-30天）30-60分、专家（>30天）60-100分；
- 功能使用广度评分：使用功能数/总功能数×30分；
- 高级功能使用频率评分：使用高级功能次数/总操作次数×20分；
- 错误率评分：(1-错误操作次数/总操作次数)×20分。

**权利要求5**：根据权利要求1所述的方法，其特征在于，所述智能图表推荐的规则包括：

- 占比分析（数据项<8）→饼图/环形图；
- 趋势分析（时间序列）→折线图/面积图；
- 对比分析（多类别比较）→柱状图/条形图；
- 关联分析（两变量关系）→散点图/气泡图；
- 分布分析（数据分布）→直方图/箱线图。

**权利要求6**：根据权利要求1所述的方法，其特征在于，还包括实时数据联动：

- 多图表联动：点击一个图表，其他图表同步更新；
- 筛选联动：筛选条件应用到所有图表；
- 高亮联动：悬停一个数据点，其他图表高亮相关数据。

**权利要求7**：根据权利要求1所述的方法，其特征在于，所述交互式数据钻取还包括：

- 面包屑导航：显示当前钻取路径；
- 返回按钮：快速返回上一级；
- 缩放动画：从点击位置放大展开。

**权利要求8**：根据权利要求2所述的系统，其特征在于，还包括：

- 动态重构引擎：根据能力评分实时调整界面；
- 过渡动画器：实现平滑过渡，避免突然变化；
- 用户偏好存储：允许用户手动切换模式。

**权利要求9**：根据权利要求1所述的方法，其特征在于，所述渐进式信息披露针对财务领域进行特殊设计：

- 核心层采用财务优先级排序：总收入→总支出→结余→预算使用率；
- 详细层包含财务特有内容：分类占比、预算vs实际对比、同比/环比变化、异常消费标记；
- 深度层包含专业财务分析：消费模式识别、财务健康评分、支出预测、优化建议。

**权利要求10**：根据权利要求1所述的方法，其特征在于，所述用户能力评估增加财务行为维度：

- 记账频率维度（占比10%）：每日记账得满分，偶尔记账得低分；
- 预算执行率维度（占比10%）：执行率>90%得满分，执行率<70%得低分；
- 财务知识测评维度（占比5%）：基于新手教程完成度和高级功能使用情况。

**权利要求11**：根据权利要求1所述的方法，其特征在于，所述智能图表推荐增加财务语义规则：

- 收支分析推荐瀑布图：展示收入、各类支出、结余的流动；
- 预算对比推荐子弹图：实际vs预算vs目标的三层对比；
- 财务健康推荐雷达图：多维度财务健康指标可视化；
- 资产配置推荐旭日图：展示资产类别的层级结构；
- 现金流推荐桑基图：展示资金来源和去向的流动。

**权利要求12**：根据权利要求1所述的方法，其特征在于，还包括财务数据的特殊处理：

- 货币精度处理：所有金额保持2位小数精度，采用货币格式显示（¥10,000.00）；
- 隐私脱敏显示：支持部分隐藏金额（如¥*,***.**），保护用户隐私；
- 财务周期感知：基于记账周期和预算周期进行时间维度分析。
