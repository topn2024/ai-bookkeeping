# 权利要求书

## 针对不稳定收入人群的自适应财务管理方法及系统

### 独立权利要求

**权利要求1**：一种针对不稳定收入人群的自适应财务管理方法，其特征在于，包括以下步骤：

S1. 收入稳定性评估步骤：
- 收集用户历史收入数据，至少6个月；
- 计算收入变异系数CV = 标准差 / 平均值；
- 基于CV判定稳定性等级：CV<0.1为高度稳定，CV≥0.5为高度不稳定；
- 综合评分 = 0.4×CV评分 + 0.3×固定收入评分 + 0.2×多样性评分 + 0.1×周期性评分；

S2. 自适应预算计算步骤：
- 根据稳定性确定可用收入比例：高度稳定90%、较稳定80%、中等稳定70%、不稳定60%、高度不稳定50%；
- 根据稳定性选择基准收入策略：高度不稳定取最小值，不稳定取25%分位数，中等稳定取中位数，较稳定/高度稳定取平均值；

S3. 收入平滑步骤：
- 创建收入缓冲账户；
- 超额收入存入缓冲账户；
- 不足收入从缓冲账户补充；
- 缓冲账户健康度 = 当前余额 / (目标收入×3) × 100%；

S4. 应急资金规划步骤：
- 计算应急月数 = 3 + (100-稳定性评分) / 10，范围3-12个月；
- 目标资金 = 月度支出 × 应急月数；
- 生成储蓄计划 = 资金缺口 / 目标期限。

**权利要求2**：一种针对不稳定收入人群的自适应财务管理系统，其特征在于，包括：

稳定性评估模块：用于计算收入变异系数和综合稳定性评分；

自适应预算模块：用于根据稳定性确定可用收入比例和月度预算；

收入平滑模块：用于管理缓冲账户和发放固定工资；

应急规划模块：用于计算应急资金目标和储蓄计划。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，所述稳定性评估的四个维度包括：

- 变异系数评分：CV<0.1为90-100分，CV≥0.5为0-29分；
- 固定收入占比评分：固定收入占比×100；
- 收入来源多样性评分：min(来源数量×10, 100)；
- 周期性规律评分：周期性收入占比×100。

**权利要求4**：根据权利要求1所述的方法，其特征在于，所述基准收入策略包括：

- 保守策略：取近6个月最小值，适用于高度不稳定；
- 稳健策略：取近6个月25%分位数，适用于不稳定；
- 平衡策略：取近6个月中位数，适用于中等稳定；
- 乐观策略：取近6个月平均值，适用于较稳定和高度稳定。

**权利要求5**：根据权利要求1所述的方法，其特征在于，所述收入平滑机制包括：

- 每月固定日期发放固定工资；
- 实际收入超过目标时，超额部分存入缓冲账户；
- 实际收入不足时，从缓冲账户补充差额；
- 缓冲资金不足时触发预警。

**权利要求6**：根据权利要求1所述的方法，其特征在于，还包括收入预测步骤：

- 趋势分析：使用线性回归计算收入趋势；
- 季节性分析：计算每月季节性因子；
- 预测计算：基准预测 + 趋势调整 × 季节性因子；
- 置信区间：预测值 ± 1.96 × 标准差。

**权利要求7**：根据权利要求6所述的方法，其特征在于，所述季节性因子考虑中国市场特点：

- 春节因子（1-2月）；
- 国庆因子（10月）；
- 年终因子（12月）。

**权利要求8**：根据权利要求1所述的方法，其特征在于，还包括动态调整步骤：

- 每月根据实际收入更新稳定性评估；
- 收入超预期时增加储蓄，不增加预算；
- 收入低于预期时动用缓冲资金，维持预算。

**权利要求9**：根据权利要求2所述的系统，其特征在于，还包括：

- 收入预测模块：预测未来3个月收入及置信区间；
- 可视化模块：展示收入趋势、稳定性评分、预算执行情况；
- 提醒模块：缓冲资金不足时发送预警。
