# 摘要

## 多模态融合智能记账方法及系统

本发明公开了一种多模态融合智能记账方法及系统，属于人工智能和财务管理技术领域。该方法包括：模态识别与处理，接收语音、图像或文本输入，分别通过ASR语音识别、OCR文字识别和票据解析、NLP语义理解进行处理；要素提取，从每种模态独立提取记账要素（金额、类别、时间、备注）并计算置信度；跨模态融合，将各模态特征向量对齐到统一空间，使用跨模态注意力机制计算权重并加权融合；结果生成，基于置信度动态加权输出最终记账结果。本发明解决了单模态输入无法充分利用多源信息、不同模态识别结果无法有效融合的问题，支持语音、拍照、文字三种便捷输入方式，多模态融合显著提升金额和类别识别准确率。

**关键词**：多模态融合；智能记账；语音识别；OCR；跨模态注意力；置信度加权
