# 权利要求书

## 基于债务收入比的智能债务管理和还款优化方法及系统

### 独立权利要求

**权利要求1**：一种基于债务收入比的智能债务管理方法，其特征在于，包括以下步骤：

S1. 债务健康评估步骤：
- 计算债务收入比DTI = 总债务余额 / 月收入；
- 计算还款比payment_ratio = 月还款总额 / 月收入；
- 划分五级健康等级：健康（DTI≤20%，还款比≤30%）、轻度负债（20%<DTI≤30%，30%<还款比≤40%）、中度负债（30%<DTI≤40%，40%<还款比≤50%）、重度负债（40%<DTI≤50%，50%<还款比≤60%）、危险（DTI>50%，还款比>60%）；
- 计算健康评分：dti_score = max(0, 100 - DTI×200)，payment_score = max(0, 100 - payment_ratio×150)，interest_score = max(0, 100 - avg_interest_rate×500)，HealthScore = 0.4×dti_score + 0.4×payment_score + 0.2×interest_score；
- 健康评估准确率>90%，响应时间<80ms；

S2. 还款策略选择步骤：
- 雪球法（Debt Snowball）：按余额升序排列债务，优先偿还余额最小的债务，获得心理激励；
- 雪崩法（Debt Avalanche）：按利率降序排列债务，优先偿还利率最高的债务，节省最多利息；
- 混合法：先用雪球法还清1-2笔小额债务获得激励（阶段1），再转向雪崩法节省利息（阶段2），平衡因子30%雪球+70%雪崩；
- 雪崩法节省利息15-30%，雪球法提前还清债务6-12个月；

S3. 智能还款计划生成步骤：
- 支付所有债务的最低还款额min_payments = Σ(debt.min_payment)；
- 根据选定策略分配额外还款金额extra = Income×0.2 - min_payments；
- 设置还款优先级：P1（最高）逾期债务、P2（高）高息债务（利率>15%）、P3（中）中息债务（利率10-15%）、P4（低）低息债务（利率<10%）；
- 生成月度还款计划，动态更新债务余额；
- 还款计划优化准确率>88%，执行率>85%；

S4. 月利息成本追踪步骤：
- 计算单笔月利息monthly_interest = balance × (annual_rate / 12)；
- 计算总月利息total_monthly = Σ(debt.monthly_interest)；
- 计算累计利息total_interest = Σ(已支付利息) + Σ(未来预计利息)；
- 计算利息占比interest_ratio = total_monthly / monthly_income × 100%；
- 对比不同策略的利息节省：saving_potential = snowball_interest - avalanche_interest；
- 利息成本透明度100%，帮助用户清晰了解债务成本；

S5. 动态调整步骤：
- 收入增加：增加还款金额，缩短还清时间；
- 收入减少：降低还款金额至最低还款，保证基本生活；
- 新增债务：重新优化还款计划，调整优先级；
- 逾期率从>20%降低到<5%，降低75%。

**权利要求2**：一种基于债务收入比的智能债务管理系统，其特征在于，包括：

健康评估模块：配置为计算DTI和还款比，评估债务健康等级，评估准确率>90%，响应时间<80ms；

策略选择模块：配置为提供雪球法、雪崩法、混合法三种还款策略，雪崩法节省利息15-30%；

计划生成模块：配置为生成智能还款计划并设置优先级，计划优化准确率>88%，执行率>85%；

利息追踪模块：配置为追踪月利息成本和累计利息，利息成本透明度100%；

动态调整模块：配置为根据收入变化调整计划，逾期率从>20%降低到<5%。

### 从属权利要求

**权利要求3**：根据权利要求1所述的方法，其特征在于，所述健康评分的具体计算为：

- dti_score = max(0, 100 - DTI × 200)，DTI越低评分越高；
- payment_score = max(0, 100 - 还款比 × 150)，还款比越低评分越高；
- interest_score = max(0, 100 - 平均利率 × 500)，平均利率越低评分越高；
- 综合评分HealthScore = 0.4 × dti_score + 0.4 × payment_score + 0.2 × interest_score；
- 综合评分范围0-100分，分数越高债务健康状况越好。

**权利要求4**：根据权利要求1所述的方法，其特征在于，所述雪球法的具体实施包括滚雪球效应：

- 按余额升序排序债务sorted_debts = sort(Debts, key=balance, ascending=True)；
- 支付所有最低还款额，剩余金额remaining = AvailableAmount - Σ(min_payments)；
- 额外还款分配给余额最小的债务target_debt = sorted_debts[0]，extra_payment = min(remaining, target_debt.balance)；
- 还清一笔债务后，释放该债务的最低还款额freed_amount = target_debt.min_payment；
- 释放的金额滚动到下一笔目标债务AvailableAmount += freed_amount；
- 每还清一笔债务显示庆祝动画和进度更新，增强用户信心。

**权利要求5**：根据权利要求1所述的方法，其特征在于，所述雪崩法的具体实施包括利息节省计算：

- 按利率降序排序债务sorted_debts = sort(Debts, key=interest_rate, descending=True)；
- 支付所有最低还款额，剩余金额分配给利率最高的债务；
- 节省利息计算：saved_interest = target_debt.balance × (target_debt.interest_rate / 12) × months_saved；
- 展示与雪球法的利息对比，计算提前还清的时间；
- 提供优化反馈，帮助用户理解利息节省的价值。

**权利要求6**：根据权利要求1所述的方法，其特征在于，所述混合法的具体实施为：

- 阶段1（前1-3个月）：使用雪球法快速还清1-2笔小额债务，获得心理激励和成就感；

**权利要求7**：根据权利要求1所述的方法，其特征在于，所述还款优先级的具体划分为：
