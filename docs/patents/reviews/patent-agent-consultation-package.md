   - 国际文献检索是否充分？
   - 是否需要补充中文数据库检索（CNIPA、CNKI）？

   - 单个专利的代理费用？
   - 从申请到授权的预计时间？
   - 是否有费用减免政策？

### 6.2 建议咨询的问题

1. **专利布局**:
   - 是否建议申请国际专利（PCT）？
   - 是否建议申请实用新型专利作为补充？
   - 是否建议申请软件著作权？

   - 哪些技术细节应该保密而非公开？
   - 是否建议申请商业秘密保护？

   - 专利的商业价值评估？
