## 🎯 立即开始

### 方案A: 使用Web检索助手（推荐）

**步骤1**: 打开Web检索助手
```bash
open scripts/search-assistant.html
```

**步骤2**: 选择专利
- 建议从P14或P18开始（风险最低，质量最高）

**步骤3**: 复制检索式
- 点击"复制"按钮，将检索式复制到剪贴板

**步骤4**: 打开数据库
- 点击"打开检索系统"按钮
- 在新标签页中打开对应数据库

**步骤5**: 执行检索
- 在数据库中粘贴检索式
- 执行检索并筛选结果

**步骤6**: 记录结果
- 返回Web检索助手
- 填写检索记录表单
- 点击"保存记录"

## 🔑 关键提示

### 数据库账号准备

**必须准备**:
- [ ] 国家知识产权局账号（免费注册）
- [ ] CNKI账号（机构账号或付费）
- [ ] 万方数据库账号（机构账号或付费）

**可选准备**:
- [ ] Google Scholar（免费，无需账号）
- [ ] Google Patents（免费，无需账号）

### 检索技巧

**扩展检索**:
- 如果结果太少（<10篇），放宽检索条件
- 如果结果太多（>1000篇），增加限制条件

**引文检索**:
- 查看高度相关文献的引用文献
- 查看高度相关文献的被引文献

**时间筛选**:
- 优先检索近5年的文献（2020-2025）
- 对于基础技术，可扩展到10年

### 相关性判断

**高度相关**:
- 技术方案高度相似
- 可能影响新颖性
- 需要详细分析

**中度相关**:
- 部分技术特征相似
- 可能影响创造性
- 需要记录差异点

**低度相关**:
- 仅技术领域相关
- 影响较小
- 简单记录即可

## 🆘 遇到问题？

### 常见问题

**Q1: 数据库无法访问？**
- 检查网络连接
- 确认账号是否有效
- 尝试使用VPN（Google Scholar）

**Q2: 检索结果太多？**
- 增加限制条件（时间、IPC分类）
- 使用更精确的检索式
- 按相关性排序，只看前50条

**Q3: 检索结果太少？**
- 放宽检索条件
- 使用同义词扩展
- 尝试不同的检索式组合

**Q4: 如何判断相关性？**
- 阅读标题和摘要
- 查看技术方案
- 对比技术特征

**Q5: Web助手记录丢失？**
- 定期导出记录到JSON文件
- 使用Python工具作为备份
- 检查浏览器localStorage
