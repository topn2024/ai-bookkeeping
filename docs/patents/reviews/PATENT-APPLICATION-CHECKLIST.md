## 五、申请流程

### 5.1 准备阶段（1-2周）

1. **整理申请人信息**
   - 确定申请人（个人/公司）
   - 准备身份证明文件
   - 确定联系地址和电话

2. **整理发明人信息**
   - 确定发明人名单
   - 准备发明人身份证明

3. **格式转换**
   - 将Markdown文档转换为Word格式
   - 调整格式符合国家知识产权局要求
   - 添加页眉、页脚、页码

4. **附图准备**
   - 转换SVG为PNG/JPG（300 DPI）
   - 补充缺失的附图
   - 确保图片质量

5. **专业审核**（强烈建议）
   - 委托专利代理人审核
   - 检查法律合规性
   - 优化权利要求书

### 5.2 提交阶段（1周）

1. **在线提交**（推荐）
   - 注册国家知识产权局专利电子申请系统账号
   - 上传申请文件
   - 在线缴费

2. **纸质提交**（备选）
   - 打印申请文件（一式两份）
   - 前往国家知识产权局或地方代办处提交
   - 现场缴费或银行转账

### 5.3 审查阶段（1-3年）

1. **初步审查**（3-6个月）
   - 形式审查
   - 明显实质性缺陷审查
   - 可能收到补正通知书

2. **公开阶段**（18个月）
   - 申请公开
   - 可以请求实质审查

3. **实质审查**（1-2年）
   - 新颖性审查
   - 创造性审查
   - 实用性审查
   - 可能收到审查意见通知书

4. **授权阶段**
   - 通过审查
   - 缴纳授权费
   - 获得专利证书

## 八、注意事项

### 8.1 申请前注意事项

1. **保密要求**
   - 申请前不要公开发明内容
   - 不要在论文、会议、网站上发表
   - 不要向无保密义务的第三方透露

2. **新颖性保护**
   - 确保发明未被他人公开
   - 检查是否有相同或相似的专利申请
   - 保留发明时间证据（研发记录、邮件等）

3. **完整性检查**
   - 确保所有文件齐全
   - 检查格式是否符合要求
   - 核对申请人和发明人信息

### 8.2 申请后注意事项

1. **及时响应**
   - 收到补正通知书后2个月内答复
   - 收到审查意见通知书后4个月内答复
   - 逾期未答复视为撤回申请

2. **费用缴纳**
   - 申请费在申请日起2个月内缴纳
   - 实质审查费在申请日起3年内缴纳
   - 授权费在收到授权通知书后2个月内缴纳

3. **年费缴纳**
   - 授权后每年缴纳年费
   - 第1-3年：900元/年
   - 第4-6年：1,200元/年
   - 第7-9年：2,000元/年
   - 第10-12年：4,000元/年
