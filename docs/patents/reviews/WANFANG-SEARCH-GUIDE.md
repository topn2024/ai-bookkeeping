**生成日期**: 2026-01-19
**目的**: 为18个专利进行现有技术检索，增强新颖性和创造性论证

---

## 一、万方数据库简介

### 1.1 数据库信息

- **网址**: http://www.wanfangdata.com.cn
- **覆盖范围**: 中国专利、学术论文、会议论文、标准、成果等
- **专利数据**: 包含中国专利全文数据库
- **更新频率**: 每周更新

### 1.2 检索优势

1. **中文检索**: 支持中文关键词检索，适合中国专利
2. **全文检索**: 可检索专利全文内容
3. **分类检索**: 支持IPC分类号检索
4. **法律状态**: 可查询专利法律状态

## 二、检索策略

### 2.1 检索步骤

1. **确定检索关键词**
   - 核心技术词
   - 同义词和近义词
   - 英文术语

2. **构建检索式**
   - 使用布尔逻辑（AND、OR、NOT）
   - 使用截词符（*、?）
   - 使用字段限定（TI、AB、CL）

3. **执行检索**
   - 初步检索
   - 调整检索式
   - 精确检索

4. **筛选结果**
   - 阅读摘要
   - 下载全文
   - 分析相关性

5. **记录结果**
   - 记录检索式
   - 记录相关专利
   - 分析技术差异

### 2.2 检索字段说明

| 字段代码 | 字段名称 | 说明 |
|---------|---------|------|
| TI | 标题 | 专利名称 |
| AB | 摘要 | 专利摘要 |
| CL | 权利要求 | 权利要求书 |
| IPC | 国际专利分类号 | IPC分类 |
| AP | 申请人 | 专利申请人 |
| IN | 发明人 | 发明人姓名 |
| AD | 申请日 | 申请日期 |
| PD | 公开日 | 公开日期 |

### 2.3 检索技巧

1. **关键词组合**:
   ```
   (关键词1 OR 同义词1) AND (关键词2 OR 同义词2)
   ```

2. **字段限定**:
   ```
   TI=(关键词1) AND AB=(关键词2)
   ```

3. **截词检索**:
   ```
   财务* (匹配：财务管理、财务分析、财务健康等)
   ```

4. **时间限定**:
   ```
   AD=[2020 TO 2025] (申请日在2020-2025年)
   ```

## 五、检索结果分析

### 5.1 相关度判断标准

**高相关**（相关度>80%）:
- 技术领域相同
- 核心技术特征相似
- 应用场景一致
- **需要详细分析技术差异**

**中相关**（相关度50-80%）:
- 技术领域相近
- 部分技术特征相似
- 应用场景部分重叠
- **需要说明技术改进**

**低相关**（相关度<50%）:
- 技术领域不同
- 技术特征差异大
- 应用场景不同
- **可作为背景技术参考**

### 5.2 技术差异分析要点

1. **核心技术特征对比**:
   - 算法原理是否相同
   - 数据结构是否相同
   - 技术参数是否相同

2. **技术效果对比**:
   - 性能指标对比
   - 准确率对比
   - 效率对比

3. **应用场景对比**:
   - 应用领域是否相同
   - 解决的问题是否相同
   - 用户群体是否相同

4. **创新点识别**:
   - 本发明的独特技术特征
   - 本发明的技术优势
   - 本发明的意外效果
