## 审查方法

### 五维审查框架

1. **维度1: 法律合规性审查** (30分)
   - 新颖性检查 (10分)
   - 创造性评估 (15分)
   - 实用性验证 (5分)

2. **维度2: 技术性审查** (25分)
   - 技术问题识别 (8分)
   - 技术手段分析 (8分)
   - 技术效果量化 (9分)

3. **维度3: 文档质量审查** (20分)
   - 说明书充分公开 (8分)
   - 权利要求清楚简要 (8分)
   - 附图完整规范 (4分)

4. **维度4: 现有技术对比** (15分)
   - 新颖性对比 (8分)
   - 创造性对比 (7分)

5. **维度5: 商业价值评估** (10分)
   - 市场需求分析 (5分)
   - 实施可行性 (5分)

- 90-100分: 授权成功率 95%+

- 70-79分: 授权成功率 70-85%
