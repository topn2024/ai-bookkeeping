### 案例2：P01-FIFO钱龄计算

**修复前**:
```
S1. 资源入池步骤：
- 接收收入事件，创建资源单元
- 记录属性：入池时间戳、资金来源、原始金额、剩余金额、资源ID
- 将资源单元按时间顺序加入FIFO资源池
```

**修复后**:
```
S1. 资源入池步骤：采用双端队列数据结构构建FIFO资源池，
接收收入事件时创建资源单元，为每个资源单元记录入池时间戳、
资金来源、原始金额、剩余金额、资源ID，将资源单元从队列尾部入队，
入队操作时间复杂度为O(1)；
```

**提升效果**:
- 明确了数据结构（双端队列）
- 增加了时间复杂度（O(1)）
- 详细说明了操作流程
