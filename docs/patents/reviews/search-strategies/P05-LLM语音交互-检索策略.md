## 1. 专利概述

### 1.1 技术领域
人工智能与语音交互技术领域，具体涉及基于大语言模型(LLM)的智能语音记账交互方法及系统。

### 1.2 核心技术方案
- **LLM意图理解**：使用大语言模型理解用户语音意图
- **多轮对话管理**：支持多轮对话补全信息
- **上下文记忆**：维护对话上下文
- **语音纠错**：自动纠正语音识别错误
- **自然语言生成**：生成自然的对话回复

### 1.3 关键技术指标
- 意图识别准确率: >92%
- 多轮对话成功率: >85%
- 语音纠错准确率: >88%
- 响应时间: <500ms
