## 5. 检索技巧

### 5.1 如果结果太多（>500条）

**调整策略**:
1. 增加限制条件，如添加"记账"或"财务管理"
2. 缩小时间范围到2020-2026
3. 使用更精确的检索式
4. 按相关性排序，只看前50条

### 5.2 如果结果太少（<10条）

**调整策略**:
1. 放宽检索条件，去掉部分AND条件
2. 使用更多同义词
3. 扩大时间范围到2010-2026
4. 分别检索各个关键词组合

### 5.3 引文检索

对于高度相关的文献:
1. 查看其引用的参考文献
2. 查看引用它的后续文献
3. 可能发现更多相关文献
