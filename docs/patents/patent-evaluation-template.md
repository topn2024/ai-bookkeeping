## 二、新颖性评估（Novelty）

### 2.1 评估标准

新颖性要求：发明在申请日前不属于现有技术

| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极高 | 无相关现有技术，技术方案完全新颖 |
| 4分 | 高 | 现有技术有类似领域，但技术方案有明显差异 |
| 3分 | 中等 | 存在相关现有技术，但关键技术特征不同 |
| 2分 | 较低 | 现有技术高度相关，仅部分特征不同 |
| 1分 | 低 | 与现有技术高度重叠，难以区分 |

## 三、创造性评估（Inventive Step）

### 3.1 评估标准

创造性要求：与现有技术相比，具有突出的实质性特点和显著的进步

| 评分 | 等级 | 标准 |
|------|------|------|
| 5分 | 极高 | 解决了技术偏见或长期未解决的技术问题 |
| 4分 | 高 | 技术方案非显而易见，需要创造性劳动 |
| 3分 | 中等 | 有一定技术贡献，但本领域技术人员可能想到 |
| 2分 | 较低 | 技术方案较为常规，创新程度有限 |
| 1分 | 低 | 是现有技术的简单组合或替换 |

| 2分 | 较低 | 仅有设计，未验证可行性 |
