# 语音记账界面原型

这是一个独立的 Flutter 界面原型，用于预览语音记账功能的新设计。

## 快速运行

### 方法一：创建新项目（推荐）

```bash
# 1. 创建新的 Flutter 项目
flutter create voice_demo
cd voice_demo

# 2. 复制原型代码
# 将 main.dart 的内容复制到 voice_demo/lib/main.dart

# 3. 运行项目
flutter run
```

### 方法二：使用 DartPad 在线预览

1. 打开 https://dartpad.dev/
2. 点击右上角 "New" -> "Flutter"
3. 将 `main.dart` 的完整内容粘贴进去
4. 点击 "Run" 按钮即可预览

## 界面状态

原型包含5个完整的界面状态：

1. **准备录音** - 中央大麦克风按钮，带脉冲动画
2. **录音中** - 显示录音时长、波形动画、红色停止按钮
3. **识别中** - 加载动画
4. **识别结果** - 显示金额、分类、备注，一键确认或编辑
5. **记账成功** - 成功动画，继续记账按钮

## 设计亮点

- **单手操作优化**：所有主要按钮都在屏幕下半部分
- **大按钮设计**：录音按钮 140px，确认按钮 56px 高度
- **流畅动画**：波形动画、成功弹出动画
- **触觉反馈**：关键操作有震动反馈
- **深色渐变背景**：现代科技感设计

## 交互流程

```
点击麦克风 → 开始录音 → 点击停止 → 显示结果 → 点击确认 → 完成
                                        ↓
                                   点击编辑 → 底部面板编辑 → 保存
```
