# 版本 2.0.3 更新说明

## 🐛 Bug修复

### 密码找回功能修复
- **修复了密码找回邮件发送问题**
  - 之前版本提示"验证码已发送"但实际没有发送邮件
  - 现在会真正发送包含6位验证码的邮件到用户邮箱
  - 验证码有效期：10分钟

### 技术改进
- 完善了前端与后端的密码重置API对接
- 添加了完整的错误处理和用户提示
- 优化了验证码验证流程

## 📝 使用说明

忘记密码时：
1. 点击"忘记密码"
2. 输入注册时使用的邮箱
3. 查收邮件中的6位验证码
4. 输入验证码并设置新密码

**注意**: 如收不到邮件，请检查垃圾邮件箱
