# 专利文档规范增量

## 修改需求

### 需求：摘要字数限制

专利摘要必须符合以下规范：

1. 摘要正文内容必须不超过300字（中文字符）
2. 摘要必须包含技术领域、技术方案要点、技术效果三要素
3. 摘要禁止包含具体参数数值，参数说明应在说明书中详述

#### 场景：摘要字数验证

**假设** 专利摘要文件已创建
**当** 检查摘要正文字数时
**那么** 字数必须≤300字
**并且** 不包含markdown标题和关键词行

#### 场景：摘要内容完整性

**假设** 专利摘要需要精简
**当** 删除详细描述时
**那么** 必须保留核心创新点
**并且** 必须包含技术效果说明
